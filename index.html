CRUD - Python/SQLite3

About the project

This project is a simple phone book that shows how to create, read, update and delete registers from a SQLite3 database.

Run

Run the file interface.py to start the software

How to use

  • To add a contact: fill Name and Phone fields and click on Save button
  • To search a contact: start typing in Type to search and the list will be filtered
  • To edit a contact: select the contact, click on Edit Button. Correct the information on Name and Phone fields, then click on Save button
  • To delete a contact: select a contact on the list and click Delete

More informations

Python files

  • interface.py - Tkinter user interface.
  • crud_python_sqlite.py - shows the querys to comunicate with database
  • back_functions.py - The communication between interface and database.
