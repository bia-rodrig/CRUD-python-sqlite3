# Python - SQLITE3 - CRUD

## Project main page: https://bia-rodrig.github.io/CRUD-python-sqlite3/

New Version of SQLite CRUD with tkinter interface

Application that shows how to:

* Create
* Read
* Update
* Delete
* Search

Data from SQLite3 Database.

## Database

Database file: db/contacts.db

Table: Contacts

Colunms:

* ID: integer
* Name: varchar(100)
* Phone: varchar(20)
